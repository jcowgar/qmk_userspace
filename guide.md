# Common Commands

| Command                                      | Description                                |
| -------------------------------------------- | ------------------------------------------ |
| `qmk flash -kb keyboardio/atreus -km jeremy` | Build and flash Atreus firmware            |
| `qmk flash -kb keychron/q8/ansi -km jeremy`  | Build and flash Keychron Q8 Alice firmware |
